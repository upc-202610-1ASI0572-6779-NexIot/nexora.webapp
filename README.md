# Nexora Web App

Main front-end for Nexora's IoT platform, developed with **Vue.js** following a **Vertical Slicing** architecture.

## 🚀 Tech Stack
- **Framework:** Vue.js 3 (Composition API)
- **Build Tool:** Vite
- **State Management:** Pinia
- **Routing:** Vue Router
- **Styles:** CSS with BEM methodology

## 🛠 Code Conventions
- **Language:** English (Code & Docs)
- **Component Naming:** PascalCase (e.g., `SensorCard.vue`)
- **Folder Structure:** Vertical Slicing by Features.
- **Commit Style:** [Conventional Commits](https://www.conventionalcommits.org/)

## 🌿 GitFlow & Branches
- `main`: Production code.
- `develop`: Integration of new features.
- `feature/*`: Development of new features.
- `hotfix/*`: Critical production fixes.
